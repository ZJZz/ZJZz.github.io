3.3 Stack of Plates

思路

题意理解

实现一个数据结构SetOfStack，当一个栈的容量超过规定容量时就新建一个栈。如果只有一个栈时，SetOfStack的Pop和Push应该和单个栈的行为一样。

Follow Up:

实现一个函数popAt,在指定的子栈进行pop操作。

分析1

在有多个栈的时候，需要知道Push和Pop的操作针对是哪个栈？

在Push的时候要考虑是否超界，在Pop时考虑是否当前栈已空，需要改变当前栈。

Hint 64

You will need to keep track of the size of each substack. When one stack is full, you may need to create a new stack.

Hint 81

Popping an element at a specific substack will mean that some stacks aren’t at full capacity. Is this an issue ?
There’re no right answer, but you should think about how to handle this.

分析2

这道题偏设计，原理很简单。

需要注意的点：

  1. Push的时候要考虑当前栈是否已满？满了的话就需要新建一个栈。
  2. Pop的时候要考虑当前的栈是否为空？空了的话就需要退回到之前的栈，如果所有的子栈都为空，就需要提示整个栈为空。
  3. popAt()需要定位到指定的栈，也就是需要一个结构来维护当前所有子栈地址。用dict还是list呢？

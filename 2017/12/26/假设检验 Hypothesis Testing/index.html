假设检验 Hypothesis Testing

To determine if a treatment has changed the value of a population parameter.

可能性级别

$\alpha$-level 用来判断某个事物可能或不可能发生的标准

如果概率小于$\alpha$-level则被视为不太可能

如果样本均的z值>z*,表明选中该样本均值的概率甚至更小

$\alpha=0.05$
$\alpha=0.01$
$\alpha=0.001$
对应下图中的绿色区域

如果落在橘色区域则认为是可能的

临界区与Z临界值

$\alpha=0.05$ Z临界值为1.65
$\alpha=0.01$ Z临界值为2.33
$\alpha=0.001$ Z临界值为3.08

均值落在任何一个临界区，表明不太可能发生，因此可能不是偶然发生的，报告最小的$\alpha$-level，表明我们的样本统计数据与总体参数非常不同，因此证明很有可能采取了任何形式的人为处理措施

例如扔飞镖，击中越小的区域表明更是由于我们的技术而不是偶然

双尾检验

假设

零假设(null hypothesis)

猜测样本均值与总体均值没有显著差异，样本均值将位于临界区外。在上图白色的区域。

对于单尾检验来说，临界区也可能在左端。

对立假设（alternative hypothesis）

猜测样本均值与总体均值存在显著差异，样本均值将位于临界区的某个位置。

我们无法证明零假设是对的，只能获取证据来拒绝零假设。

拒绝零假设失败。

拒绝零假设成功

置信区间并非是分析经过干预后的数据的最佳方式，当数值没有真正现实意义的时候。这是假设检验就派上用场。
